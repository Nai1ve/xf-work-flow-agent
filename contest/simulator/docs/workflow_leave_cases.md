## Gold Trajectory 结构

所有案例包含5个步骤：

1. `user.get_info` - 查询当前用户信息
2. `workflow.catalog` - 搜索请假流程
3. `workflow.schema` - 获取流程 schema
4. `workflow.search_person` - 搜索审批人
5. `workflow.save` - 保存请假申请

## 必填字段

- `applicant` - 申请人 ID
- `applicant_no` - 申请人工号
- `start_time` - 开始时间 (YYYY-MM-DD HH:MM)
- `end_time` - 结束时间 (YYYY-MM-DD HH:MM)
- `leave_type` - 请假类型代码
- `reason` - 请假原因代码
- `approver` - 审批人 ID
- `duration` - 时长（小时）
